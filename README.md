# Palo Alto Firewall Interview Questions 🚀

A comprehensive, interactive web-based guide for Network Security Engineers preparing for Palo Alto Networks interviews. This project categorizes 40 essential questions into three difficulty levels: Easy, Medium, and Hard.

## 🌟 Features
- **40 Detailed Questions & Answers**: Based on real-world interview scenarios.
- **Categorized Difficulty**: Divided into Level 1 (Basics), Level 2 (Intermediate), and Level 3 (Advanced).
- **Interactive UI**: Toggle answers to test your knowledge.
- **Search Functionality**: Quickly find topics like "HA", "NAT", or "GlobalProtect".
- **Mobile Responsive**: Study on the go from your phone or tablet.

## 🛠️ Built With
- HTML5
- CSS3 (Dark Mode / Palo Alto Theme)
- Vanilla JavaScript

## 📖 How to Use
1. Select a difficulty level from the top tabs.
2. Click on a question to reveal the answer.
3. Use the search bar to filter for specific technical terms.
